Example: Mobile and desktop

Want your columns to display differently on desktop? You can add -l to the end of the classes to change the width for columns for desktop screens.

.w-100 .w-two-thirds-l
.w-50 .w-third-l
.w-50 .w-third-l
.w-50 .w-third-l
.w-50 .w-third-l

Example: Mobile, tablet, desktop

You can build on the previous example by adding on -ns or -m to the classes for tablets.

.w-100 .w-50-ns .w-50-m .w-two-thirds-l
.w-50 .w-third-l
.w-50 .w-third-ns .w-third-m
.w-50 .w-third-ns .w-third-m
.w-50 .w-third-ns .w-third-m

Example: Column wrapping

If the columns taking up more than 100% width are placed in a single row, the extra columns will wrap onto a new line.

.w-90

.w-30

Since 90 + 30 > 100, this 30% width div gets wrapped onto a new line.

.w-50

Subsequent columns continue alone the new line.
