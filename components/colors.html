To use the colors, simply include the color name in the class list for a block of text, e.g: 

	<p class="red">this text will be red<p>

Custom Colors

You can easily add your own custom colors in your CSS file.
We have done this with our brand color. e.g: 

.dwyl-bg-mint {
  background-color: #4BC0A9;
}
And then we can use the background color anywhere in our site/app: Start Today!

For more on "skins" and accessible color combinations, see: tachyons.io/docs/themes/skins/
