Condensed table

Cut the padding in half to have a more compact table. Since padding increases by multiples of 2 in tachyons, this is accomplished by going down by one on all of your padding. For example, if you were using pv3, you can halve this by instead using pv2.

EXAMPLE

# First Name Last Name Username
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter 
<table class="w-100 collapse ph1">
  <thead>
    <tr class="bb b--silver">
      <th class="tl pv2 ph1">#</th>
      <th class="tl pv2">First Name</th>
      <th class="tl pv2">Last Name</th>
      <th class="tl pv2">Username</th>
    </tr>
  </thead>
  <tbody>
    <tr class="bb b--moon-gray">
      <th class="tl ph1">1</th>
      <td class="pv2">Mark</td>
      <td class="pv2">Otto</td>
      <td class="pv2">@mdo</td>
    </tr>
    ...
    <tr>
      ...
    </tr>
  </tbody>
</table>

Contextual classes

Add contextual colouring to table rows or individual cells. For highlighting whole rows, add the background colour on the tr tag. For highlighting a single cell, add the background colour to a th or td tag.

Class Description
.bg-light-gray Applies the hover color to a particular row or cell
.bg-light-green Indicates a successful or positive action
.bg-light-blue Indicates a neutral informative change or action
.bg-light-yellow Indicates a warning that might need attention
.bg-washed-red Indicates a dangerous or potentially negative action
