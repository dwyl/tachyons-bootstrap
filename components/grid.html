Example: Fluid container

Grid layouts can have a fixed width by using any of:

  • w1
  • w2
  • w3
  • w4
  • w5
in the outer wrapping div.

By using w-100 on the outer div, the grid will have a full-width layout.

Example: Column wrapping

If the columns taking up more than 100% width are placed in a single row, the extra columns will wrap onto a new line.

.w-90

.w-30

Since 90 + 30 > 100, this 30% width div gets wrapped onto a new line.

.w-50

Subsequent columns continue alone the new line.

<div class="w-100 h3">
  <div class="fl w-90 pa2 bg-dwyl-teal ba b--dwyl-dark-teal">.w-90</div>
  <div class="fl w-30 pa2 bg-dwyl-teal ba b--dwyl-dark-teal">
    <p>.w-30</p>
    <p>Since 90 + 30 > 100, this 30% width div gets wrapped onto a new line.</p>
  </div>
  <div class="fl w-50 pa2 bg-dwyl-teal ba b--dwyl-dark-teal">
    <p>.w-50</p>
    <p>Subsequent columns continue alone the new line.</p>
  </div>
</div>

Nesting columns

To nest content, you will need to add an extra div with the class cf. Nested rows should still use columns that add up to 100% or less.

Level 1: .w-100

Level 2: .w-50
Level 2: .w-50
